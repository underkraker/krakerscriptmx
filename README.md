# 🐲 KRAKER MASTER - VPS Management Panel v2.0 🐲

Bienvenido al panel de gestión de VPS más avanzado y optimizado para **Gaming** y **Ultra-Stealth**. Este sistema ha sido diseñado desde cero para ofrecer una experiencia fluida, robusta y profesional en cualquier servidor Linux (Ubuntu/Debian).

## 🚀 Características Principales

*   **Elite Dashboard:** Monitor en tiempo real de CPU, RAM, IP y Uptime.
*   **Protocolos de Última Generación:**
    *   Xray REALITY (VLESS) - Vision Flow para máxima invisibilidad.
    *   Hysteria v2 - Potencia UDP para gaming extremo.
    *   Trojan/VMess WS + TLS - Compatibilidad total con inyectores.
    *   Shadowsocks WS + TLS - Rapidez y simplicidad.
    *   Extreme SlowDNS - Conexión garantizada sin saldo.
*   **Optimizaciones:**
    *   Kernel BBR activado para evitar congestión.
    *   Prioridad alta de CPU para servicios UDP (BadVPN).
    *   Manejo automático de Firewall (UFW/Iptables).
*   **Modularidad:** Código 100% limpio y centralizado a través de `utils.sh`.

## 📦 Instalación Rápida (One-Step)

Para instalar o actualizar el panel, ejecuta el siguiente comando en tu terminal como usuario **root**:

```bash
bash <(curl -fsSL https://raw.githubusercontent.com/underkraker/krakerscriptmx/main/install.sh)
```

## 🛠️ Comandos Post-Instalación

Una vez instalado, puedes acceder al panel en cualquier momento simplemente escribiendo:

```bash
kraker
```

## 📋 Requisitos del Sistema

*   **S.O:** Ubuntu 20.04/22.04 LTS (Recomendado) o Debian 11/12.
*   **Hardware:** Mínimo 512MB RAM (1GB recomendado para Xray).
*   **Permisos:** Usuario Root.

---

## 🔒 Seguridad y Privacidad

El panel genera llaves de seguridad únicas (UUIDs, PrivateKeys, Passwords) en cada instalación. No almacena logs de usuario en el servidor, garantizando la privacidad absoluta de tus conexiones.
